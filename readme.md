# Entry-Fontter
Entry-Fontter 확장 프로그램은 엔트리 작품 글상자에 커스텀 웹폰트 적용을 쉽게 만들어 줍니다.


[제작자 프로필 링크](https://playentry.org/profile/67e8e2b5eb06d385f08208ee/)


## 설치 방법
1. 현재 페이지 상단의 Code > Download Zip 선택
2. 다운로드된 Zip 파일의 압축을 해제
3. [Chrome 확장 프로그램 페이지](chrome://extensions/)로 이동
4. 우측 상단 '개발자 모드' ON
5. 좌측 상단 '압축 해제된 확장 프로그램 설치' > 2번의 폴더 내부의 build 폴더 선택


## 사용 방법
#### 만들고 있는 작품에 웹폰트 적용하는 법
1. 작품 만들기 페이지에서 글상자 오브젝트 선택
2. 확장 프로그램 열기
3. 웹폰트 URL과 폰트 이름 (정확하게 입력해야 함)
    * 이때 URL은 CSS 코드 (@import, @font-face 등)이어야 함
    * 모르겠으면 AI로 물어보면 대부분은 알려주더라고요
4. 다른 사람이 사용할 수 있게 작품 소개나 댓글에 사용한 웹폰트 URL 남겨놓는것을 추천

#### 작품 보기 페이지에서 작품에 사용되는 웹폰트 적용시키는 법
1. 그 작품에 사용되는 웹폰트 URL을 확장 프로그램 팝업에 입력
2. 아래 목록에 새로 입력한 폰트가 적용된 것을 확인하고 새로고침

### 참고 사항
* 이 확장 프로그램은 생성형 AI를 부분적으로 사용하여 개발되었습니다.
